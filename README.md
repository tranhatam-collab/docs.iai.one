# DOCS.IAI.ONE

DOCS.IAI.ONE is the temporary public documentation gate for the wider IAI system.

The current version is a holding page that introduces the documentation portal and routes users to the active system layers:

- `home.iai.one`
- `iai.one`
- `flow.iai.one`

## Current Purpose

This repo provides a lightweight static site for:

- announcing the documentation portal
- preserving the public docs domain
- routing users to the current core system surfaces
- preparing for the official documentation launch later
