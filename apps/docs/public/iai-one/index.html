iai.one is the public constitutional root of the ecosystem. In the live snapshot on March 28, 2026, it redirects to www.iai.one and appears to be served through Wix.

Constitutional Role

Public Narrative

Define the idea, scope, and public promise of the IAI ecosystem in plain language.

Charter Layer

Hold the constitutional framing: why the system exists, what principles guide it, and what it claims to be building.

Entry Signal

Send users into the right next surface instead of trying to be the product, the portal, and the docs all at once.

What The Root Owns

Owns

  • Charter, positioning, and public constitutional language.
  • Clear routing into home.iai.one, docs.iai.one, and product surfaces.
  • Trust and narrative framing for founders, partners, and first-time visitors.

Does Not Own

  • Detailed technical docs.
  • Live product operations.
  • API contracts, deployment instructions, or runtime behavior.

Minimum Links The Root Should Always Expose

  • home.iai.one as the portal entry.
  • Service Registry for the full ecosystem map.
  • Getting Started for users and builders.
  • Owner Checklist for internal alignment on help and docs obligations.
