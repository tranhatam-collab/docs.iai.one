Common Questions

Short answers for the most common platform questions right now.

Answers

How do I create a flow?

Open flow.iai.one, create a flow, add input, logic, and output nodes, then run the execution.

Is it free?

The current documentation describes an MVP-stage platform. Pricing and packaging are not finalized in this docs set yet.

How is security handled?

The current default is strict CSP, no tracking, no analytics, no external scripts, and Cloudflare-native deployment.

Where does app.iai.one fit?

app.iai.one is the operator-facing surface for review, dashboards, and intervention. It is not the workflow builder itself.

What is missing today?

The platform still needs a more standardized API layer, auth layer, UI error handling, and a formal SDK release.
