Planned Infra Surface

cdn.iai.one

cdn.iai.one is intended to be the Cloudflare CDN surface for the ecosystem. As of March 28, 2026, the public host is not resolving in DNS, so this page documents launch intent and pre-publication requirements rather than a live endpoint.

Expected Purpose

Static Assets

Serve versioned assets that should be cacheable and predictable across docs, app, and supporting surfaces.

Public Bundles

Hold downloadable artifacts, media, or SDK bundles that should not live directly inside product repos.

Cache Policy

Make TTL, invalidation, and asset-versioning policy explicit instead of hiding it behind ad hoc links.

Do Not Treat As Live Yet

Current state

Because the hostname does not resolve publicly, product pages should not expose cdn.iai.one as a real user-facing link yet. Docs should describe it as planned infrastructure only.

Launch Checklist

  • DNS live and verified publicly.
  • Owner team named for cache policy and purge workflow.
  • Documented path taxonomy for assets, bundles, and media.
  • Versioning rules so clients know when URLs are immutable versus replaceable.
  • Security and privacy review for any public files or signed access patterns.
